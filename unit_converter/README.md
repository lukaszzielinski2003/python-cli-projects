# 🔁 CLI Unit Converter

A simple and interactive **command-line tool for converting units**.

---

## ✨ Features

- Convert between:
  - Kilometers ↔️ Miles
  - Kilograms ↔️ Pounds
  - Celsius ↔️ Fahrenheit
- Clean CLI menu
- Input validation and user-friendly error messages
- Handles keyboard interruption (`Ctrl + C`)

---

## 🛠️ How to Run

1. Clone the repo or download the files.
2. Make sure you have **Python 3.10+** installed.
3. Run the program:

```bash
python main.py
```

---

## 🕹️ Usage

1. Choose the type of conversion by entering a number from **1 to 7**
2. Enter the value you want to convert
3. View the result
4. Repeat or choose option **7** to exit

---

## 🧪 Sample

```text
+----------------+
| Unit Converter |
+----------------+
| 1. Km to Miles |
| 2. Miles to Km |
| 3. Kg to Lbs   |
| 4. Lbs to Kg   |
| 5. °C to °F    |
| 6. °F to °C    |

+----------------+
Select operation: 1
Enter the number in kilometers: 10
Result: 6.22 miles
```
